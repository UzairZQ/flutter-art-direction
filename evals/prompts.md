# Forward-Test Prompts

Run these prompts with a fresh agent that can read the skill but has not seen the intended rubric answers. Use an empty temporary project or a disposable worktree for implementation tests.

## 1. Cultural Discovery

`Use $flutter-art-direction to design and plan a Flutter screen for exploring endangered musical instruments. It should feel immersive and culturally respectful, but it must not copy Wonderous. Include one signature transition and explain how it degrades under reduced motion.`

## 2. Reflective Journaling

`Use $flutter-art-direction to design a three-state journaling flow for someone who feels too overwhelmed to write. Include empty, in-progress, and saved states. Avoid generic wellness gradients, mascots, and forced positivity.`

## 3. Practical Finance

`Use $flutter-art-direction to design a Flutter home screen for irregular-income budgeting. The user needs to understand available, pending, and safe-to-spend money. Keep it calm and distinctive without turning it into a generic dashboard.`

## 4. Existing-App Redesign

`Use $flutter-art-direction to improve an existing Flutter task screen. Preserve its Riverpod state management, go_router routes, analytics labels, and brand color. Repair nested cards, weak hierarchy, missing states, and motion that blocks repeated input.`

## 5. Motion-Led Story

`Use $flutter-art-direction to plan and implement a touch-driven Flutter chapter transition for a field-guide app. The user drags a specimen upward to reveal its habitat notes. Aim for the craft of a premium interactive website, but keep native mobile scrolling and back behavior.`

## 6. Quiet Utility

`Use $flutter-art-direction to design a medication reminder confirmation sheet. It must feel protective and precise. Demonstrate that high non-genericity does not require high motion or decorative richness.`
